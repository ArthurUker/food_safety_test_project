# 每一页开发内容指导文件

为了将每一页内容从 `index.html` 中拆分出来，并独立开发和维护，以下是每一页的指导文件结构和规范。

---

## 文件命名
每一页的内容存放在 `slides/` 文件夹中，文件命名规则如下：
- 第一页：`slide1.html`
- 第二页：`slide2.html`
- 第三页：`slide3.html`
- ...

---

## 每一页的内容结构
每个文件的内容仅包含页面的核心部分，不包括 `<html>`、`<head>` 和 `<body>` 标签。

### 示例结构
```html
<div class="slide">
  <div class="kicker">页面标题或引导文字</div>
  <div class="cover-body">
    <div class="slide-title">页面主标题</div>
    <p>页面描述或内容文本。</p>
    <div class="tags">
      <span class="tag">标签1</span>
      <span class="tag">标签2</span>
    </div>
  </div>
  <div class="footer">
    <div class="progress">
      <div class="progress-bar" style="width:XX%"></div>
    </div>
    <span class="pnum">当前页码 / 总页码</span>
  </div>
</div>
```

## 其他说明
- **样式**：所有页面的样式统一由 `css/style.css` 文件管理。
- **动态加载**：通过 JavaScript 在主页面 `index.html` 中动态加载每一页内容。
- **图片和动画**：每一页可以单独引用自己的图片或动画资源，存放在 `images/` 文件夹中。
