# 每一页开发内容指导文件（v2 深化版 - 28章制度体系）

> **版本说明**：本文档为深化版，将原有20页扩展为28页，新增8个关键制度章节，每页内容深度提升3-5倍。依据国市监食检规〔2023〕1号 + ISO 17025 实验室认可体系制定。

---

## 文件组织
- `slides/`：28个HTML幻灯片片段
- `index.html`：演示主页（动态加载幻灯片）
- `scripts/gen_slides_v2_expanded.py`：新版生成脚本

---

## 文件命名
每一页的内容存放在 `slides/` 文件夹中，文件命名规则：
- 第一页：`slide1.html`
- 第二页：`slide2.html`
- ...

---

## 每一页的内容结构
每个文件仅包含页面核心 HTML，不含 `<html>`、`<head>`、`<body>` 标签。

---

## 核心特色

| 维度 | 原20页 | 深化版28页 | 深化倍数 |
|------|--------|----------|---------|
| 供应链管理 | 无 | Slide 21 完整制度 | ∞ |
| 人员健康 | 部分 | Slide 22 独立制度 | 5× |
| 设施管理 | 基础 | Slide 23 详细分区 | 8× |
| 方法确认 | 无 | Slide 24 5步流程 | ∞ |
| 问题处理 | 基础 | Slide 25 五级分类+纠正 | 10× |
| 内部审核 | 无 | Slide 26 ISO 体系 | ∞ |
| 事件响应 | 基础 | Slide 27 四级分级 | 15× |
| 文件管理 | 无 | Slide 28 版本控制 | ∞ |
- **样式**：所有页面样式统一由 `css/style.css` 文件管理（16:9, 1600×900）
- **动态加载**：通过 `js/main.js` 在 `index.html` 中动态加载每页内容
- **生成脚本**：`scripts/gen_slides.py` 可重新生成所有幻灯片
- **快检定位**：所有页面遵循"快检用于风险筛查，不替代实验室检验"原则
