## 一、项目概述

### 1.1 项目背景
本项目是一套面向**食品安全推广场景**的静态网页幻灯片演示系统，内容依据国市监食检规〔2023〕1号文件（市场监管总局关于规范食品快速检测使用的意见）进行设计，旨在向市场监管部门、农贸市场、商超餐饮、学校团餐等客户，推广"食源性病原体微流控核酸快速筛查解决方案"。

### 1.2 产品定位
- **核心卖点**：食品安全风险快速筛查能力（非确证检测）
- **技术载体**：微流控核酸检测芯片（VH Plus系列）
- **应用场景**：日常监管、专项整治、重大活动保障、"你送我检"民生服务
- **推广路径**：政策合规定位 → 示范点应用 → 第三方验证 → 项目制销售 → 政府采购

### 1.3 目标受众
| 受众 | 使用场景 |
|------|---------|
| 销售团队 | 客户拜访、方案汇报、招投标演示 |
| 市场监管部门 | 政府采购论证、项目立项汇报 |
| 农贸市场/商超 | 食品快检室建设方案沟通 |
| 学校/团餐机构 | 食源性风险排查项目推介 |

## 四、技术架构说明

### 4.1 核心设计思路
- **纯静态**：无需服务器，直接浏览器打开 `index.html` 即可运行
- **等比缩放**：以 `1600×900px` 为设计基准，通过 CSS `scale` 等比适配任意分辨率屏幕
- **无框架依赖**：原生 HTML + CSS + JavaScript，零外部依赖
- **关注点分离**：HTML / CSS / JS 三文件各司其职，互不耦合

### 4.2 文件职责说明

| 文件 | 职责 |
|------|------|
| `index.html` | 仅负责页面结构与内容，不含任何样式和脚本逻辑 |
| `css/style.css` | 全部视觉样式，含 Reset、布局、组件、动画、色彩 |
| `js/main.js` | 全部交互逻辑，含缩放适配、翻页、键盘、触屏 |

### 4.3 缩放适配机制
```javascript
// js/main.js
function resize() {
  const scale = Math.min(window.innerWidth / 1600, window.innerHeight / 900);
  const offsetX = (window.innerWidth  - 1600 * scale) / 2;
  const offsetY = (window.innerHeight - 900  * scale) / 2;
  scaler.style.transform = `translate(${offsetX}px, ${offsetY}px) scale(${scale})`;
}
```
> 取宽高缩放比的较小值，保证内容完整展示，两侧留空由背景色填充。

### 4.4 幻灯片切换机制
- 所有幻灯片默认 `display:none`，激活页添加 `.active` 类变为 `display:flex`
- 切换时移除旧页 `.active`，添加新页 `.active`，同时触发 `fadeUp` 入场动画
- 当前页索引由变量 `cur` 管理

### 4.5 交互方式支持
| 交互方式 | 操作 |
|---------|------|
| 鼠标点击 | 页面底部"上一页 / 下一页"按钮 |
| 键盘快捷键 | `→` / `Space` / `PageDown` 下一页；`←` / `PageUp` 上一页 |
| 键盘快捷键 | `Home` 跳转第一页；`End` 跳转最后一页 |
| 键盘快捷键 | `1`–`9` 数字键直接跳转对应页（v1.1 新增） |
| 键盘快捷键 | `F` 键切换全屏 |
| 触屏滑动 | 左滑下一页，右滑上一页（阈值 50px） |

### 5.2 字体体系
- **字体栈**：`"Microsoft YaHei", "PingFang SC", "Segoe UI", Arial, sans-serif`
- **大标题**（封面）：42px / 800
- **页内标题 .sm**：36px / 800
- **Kicker 标签行**：18px / 700 / 绿色
- **卡片标题 h3**：19px / 蓝色
- **正文 p**：18px / 行高 1.6
- **列表项 li**：17px / 行高 1.55
- **表格内容**：15–16px

### 5.3 左侧色条
每张幻灯片通过 `::before` 伪元素在最左侧渲染一条 `14px` 宽的渐变竖条（深蓝→绿色），作为品牌识别元素。

## 七、已知问题与注意事项

1. **S14 流程条（10步）**：在小屏幕下 `.fstep` 步骤可能偏窄，文字溢出风险，建议后续对 10 步以上流程单独处理字号。
2. **表格页内容密度**：S05、S07、S10、S13 均为表格布局，内容行数较多时可能撑破 900px 高度，需控制行数或缩小字号。
3. **字体回退**：非 Windows/macOS 环境下中文字体效果依赖系统字体，暂未引入 Web Font。
4. **本地文件访问**：部分浏览器对本地 `file://` 协议加载外部 CSS/JS 有安全限制，建议使用 VS Code Live Server 插件或任意本地 HTTP 服务器预览。
